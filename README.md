# Design Doc Self Generator

Self-generates design doc.

#### Getting Started

##### Running

```
docker compose up
```

##### GraphQL Playground

[http://localhost:8080/graphql](http://localhost:8080/graphql)


##### Features

Contains the following features:
- [x] Pulls GitHub Data.
- [x] Env Variables.
- [x] Config.
- [x] Auth.
- [x] JWT.
- [x] GraphQL.
- [x] PostgreSQL.
- [x] Type Safe SQL.
- [x] Docker Compose.
- [x] Live reload.
- [x] Admin.
- [x] Unit tests.
